# Python - ToolBox

This repository is a personal learning workspace that contains examples, notes, and small projects built while learning various Python packages. Each package is organized separately with focused scripts demonstrating core concepts and common use cases.

---

## Packages Covered So Far

### 1. QR Code
- Generating QR codes from text and URLs
- Saving QR codes as image files
- Customizing size, color, and error correction

Folder: `QR/`

---

### 2. CSV
- Reading CSV files
- Writing and updating CSV data
- Working with rows, columns, and headers

Folder: `CSV/`

---

### 3. Pillow (PIL)
- Image loading and saving
- Image resizing, cropping, and rotation
- Image format conversion and basic manipulation

Folder: `Pillows/`
